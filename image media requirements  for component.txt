🎨 IMAGE SPECIFICATIONS & GUIDELINES

โลโก้และแบรนด์

```blade

<img src="{{ asset('images/logo/logo-main.png') }}" 
     alt="Blife Platform" 
     width="120" height="40">

{{-- โลโก้ไอคอน --}}
<img src="{{ asset('images/logo/logo-icon.png') }}" 
     alt="Blife" 
     width="32" height="32">
```

ข้อกำหนด:

· รูปแบบ: PNG with transparent background
· สี: ตาม brand guideline
· ขนาด: ตามที่ระบุในแต่ละตำแหน่ง

ข้อกำหนด:

· รูปแบบ: PNG/SVG
· สไตล์: Outline หรือ Flat design
· สี: Consistent color scheme
· ขนาด: 24x24px, 32x32px, 48x48px


🚀 DEPLOYMENT INSTRUCTIONS

ขั้นตอนการเพิ่มภาพ:

1. สร้างโฟลเดอร์ ตามโครงสร้างด้านบน
2. เตรียมภาพ ตามขนาดและรูปแบบที่กำหนด
3. อัพโหลดภาพ ไปยังโฟลเดอร์ที่เหมาะสม
4. ทดสอบ การแสดงผลบน devices ต่างๆ

คำสั่ง Laravel สำหรับจัดการภาพ:

```bash
# สร้าง symbolic link สำหรับ storage
php artisan storage:link

# Clear cache หลังจากเพิ่มภาพใหม่
php artisan config:clear
php artisan cache:clear
```

ตัวอย่างการตั้งค่าใน config/filesystems.php

```php
'disks' => [
    'public' => [
        'driver' => 'local',
        'root' => storage_path('app/public'),
        'url' => env('APP_URL').'/storage',
        'visibility' => 'public',
    ],
    
    'images' => [
        'driver' => 'local',
        'root' => public_path('images'),
        'url' => env('APP_URL').'/images',
        'visibility' => 'public',
    ],
],
```

ระบบภาพนี้ถูกออกแบบมาเพื่อรองรับการแสดงผลที่คมชัดบนทุกอุปกรณ์ พร้อมการโหลดที่รวดเร็วและประสิทธิภาพการแสดงผลที่ดีที่สุด
